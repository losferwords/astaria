# Хангор

Бог-кузнец, связанный с [[Урист Китин|Уристом Китином]] и [[Великая Кузня|Великой Кузней]].

В [[Глава 088 - Сон без конца|главе 88]] он рассказывает Уристу о планах богов на конец света: покинуть Астарию с помощью Великой Кузни, завершить работы с помощью [[Калипсо]] и не вмешиваться в [[Кузница Плоти|Кузницу Плоти]] без необходимости.
