# Озеро Сильвиан

![[Assets/Images/Silvian_Lake.jpg]]

## Основной текст

Крупное пресноводное озеро в северной части [[Лес Иомар|леса Иомар]]. Считается священным местом для всех [[Друид|Друидов]] [[Иомар|Иомара]], так как по их верованиям именно на берегах этого озера обитает богиня [[Церунна]].
