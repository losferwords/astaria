# Индульф мак Торн

![[Assets/Images/Indulf_mac_Thorn.jpg]]

## Основной текст

Действующий верховный [[Горец]] из клана Торн, занимающий высшее положение среди всех [[Громовые Кланы|Громовых Кланов]]. Де-факто является лидером этого государства.
Индульф один из старейших представителей народа [[Гойдаир]]. Из них 40 лет он является верховным [[Горец|Горцем]].

У Индульфа есть сестра-близнец по имени [[Элеонора мак Торн|Элеонора]].

На своём [[Рунштоирм|Рунштоирме]] Индульф разделил силу грозы лишь со своей сестрой [[Элеонора мак Торн|Элеонорой мак Торн]].
