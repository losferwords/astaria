# История Астарии

Данная хронология представляет собой лишь исторические вехи, задокументированные в различных архивах существующих на данный момент государств [[Астария|Астарии]]. Некоторые данные могут быть искажены или приукрашены летописцами.

## События

```dataview
TABLE year AS "Год", endingYear AS "До", timeline_category AS "Категория", significance AS "Значимость"
FROM "Хронология/События" OR "Энциклопедия/События"
WHERE timeline = true
SORT number(year) ASC
```
