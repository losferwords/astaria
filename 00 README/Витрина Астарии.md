## Что открыть в Obsidian

1. Откройте эту страницу в Reading view.
2. Включите Graph view и посмотрите, как узлы начали соединяться.
3. Откройте [[Астария - карта связей.canvas|Canvas-карту связей]].
4. Установите community plugin `Dataview`, чтобы ожили таблицы.
5. Установите community plugin `Obsidian Leaflet`, чтобы проверить [[Карта Астарии]].
