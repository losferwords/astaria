# Храм Дракона

![[Assets/Images/Dragon_Temple.jpg]]

## Основной текст

Священное место для всего народа [[Джу]] и [[Империя Ланг-Ан|Империи Огня]]. Храм считается местом обитания [[Дракон Ланг-Ан|Великого Дракона]] и жильём для сотен монахов, которые служат Императрице. Кроме того этот храмовый комплекс служит плацдармом для подготовки [[Аватар|Аватаров]] - избранных воинов [[Дракон Ланг-Ан|Ланг-Ана]].
